선택자

특정 html 요소를 선택하고자 할 때 사용하는 방식
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬 수 있음

선택자 종류

  • 전체 선택자　: *
  • 태그 선택자　: 태그명
  • 아이디 선택자: #아이디명
  • 클래스 선택자: .클래스명
